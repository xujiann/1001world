# 1001 世界 · 1001 World

一个纯前端的开放世界小游戏:漫步在一座收藏小镇里,亲眼看到 1001×1001 系列的真实藏品。

**玩法**
- `WASD` / 方向键移动,`E` / 空格互动,`J` 打开图鉴,`Esc` 关闭
- 手机:左半屏拖出虚拟摇杆,右下按钮互动
- 中央广场的路牌可快速旅行
- 每看一件藏品自动收进「探索图鉴」(localStorage 持久化)

| 1001 酒馆 | 1001craft(144 款) | 吧台点酒 + 酒桶,风格化酒杯按风格配色 |
| 深蓝水族馆 | 1001fish(1001 种抽 120) | 发光鱼缸 + 游鱼动画 |

**文件**
- `index.html` / `style.css` / `game.js` — 游戏本体(零依赖,Canvas 2D)
- `world-data.js` — 由 `_gen.mjs` 从各姊妹项目数据集自动抽取生成
- `serve.mjs` — 本地预览:`node serve.mjs`(默认 8123)

**重新生成数据**:`node _gen.mjs`(需本机存在各 1001 项目源码)
